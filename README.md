#### Proje Yapısı

- `index.html`: Ana HTML dosyanız.
- `style.css`: Tüm stillerinizin bulunduğu CSS dosyası.
- `assets/`: Projede kullanacağınız tüm resim ve diğer statik dosyalar sizin için önden dışarı aktarıldı.
